fetch() → {Promise.<(result|error)>}

Source:
  • methods/fetch.js, line 20
Send POST request to E-Com Plus Search API and returns promise resolved with search result.
Example
// Run search request
search.fetch()
  .then(result => {
    console.log(result.took)
    console.log(`${result.hits.total} items found:`)
    console.log(result.hits.hits)
  })
  .catch(error => {
    console.error(error)
    if (error.response) {
      console.log(error.response)
    }
  })
Returns:
Type
